## Introduction
**CMSIS** Cortex Microcontroller Software Interface Standard, a vendor-independent hardware abstraction layer for the Cortex-M processor series and defines generic tool interfaces.

### Features
- Thread
- Semaphore
- Mutex
- Message
- Timer
