# 点击 “更正文档”

开发者可以点击待修改文档上方的“更正文档”字样，页面会自动跳转到该文档所在gitee仓库的位置，并处于编辑状态。

# 修改文档
开发者可以直接在文档的编辑框中修改文档。

# 提交审核
文档修改完成后点击页面下方的“提交审核”按钮完成提交。
